Display one single photo from your WordPress blog or another WordPress-based blog on your sidebar with fancy "film like" frame.

== Description ==

A photo image on your sidebar with fancy film like frame. You can bring recent photo from your blogs. The photo source should be WordPress -based blog with default-based theme.

== Installation ==

Place the widget_single_photo folder in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!

== Frequently Asked Questions ==

= Can I bring a photo from my own other blog? =

Yes, you can specify any URL you want, but this program designed for 'photo blog' so in some case, the photo will overflow its frame.

= I'm good at CSS. What is the content id name? =

The photo and its frame belongs &gt;div id=”widget_single_photo”&lt; so you can modify your site's CSS file to decorate it.
